## 감정 기록 선택 도구 
감정 기록 시스템은 사용자가 자신의 일상 감정을 간편하게 기록하고 관리할 수 있는 웹 애플리케이션이다.
이 시스템은 직관적인 UI와 간단한 조작을 통해 누구나 쉽게 사용할 수 있으며, 감정 데이터를 실시간으로 저장하고 확인할 수 있다.
주요 기능 설명
UX 향상 (하이브리드 기록 방식):
빠른 기록: 텍스트 입력창이 비어있을 때 이모지 버튼을 누르면 즉시 Firestore에 저장됩니다. 바쁜 순간에 유용합니다.
상세 기록: 이모지를 선택한 후 메시지를 입력하고 아래 '저장하기' 버튼을 누르면 메시지와 함께 저장됩니다.
노란색 테마 디자인:
요청하신 노란색 계열의 그라데이션과 부드러운 라운드(rounded-[2.5rem]) 처리를 통해 따뜻하고 현대적인 느낌을 주었습니다.
사용자 피드백을 위해 버튼 클릭 시 크기가 줄어드는 애니메이션과 선택된 버튼의 강조 효과를 넣었습니다.
실시간성:
onSnapshot 리스너를 통해 본인이나 타인이 데이터를 추가/삭제하면 새로고침 없이 리스트가 즉시 갱신됩니다.
최신 기록이 항상 맨 위에 표시되도록 Firestore의 serverTimestamp를 사용하여 정렬했습니다.
반응형 레이아웃:
Tailwind CSS의 유틸리티 클래스를 사용하여 모바일 화면에서는 꽉 차게, PC 화면에서는 적절한 크기의 카드로 보이도록 설정했습니다.
